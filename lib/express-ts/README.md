# Development

1. Clone project

   ```bash
   git clone git@bitbucket.org:iamruin/mms-bounce-recovery.git
   mms-bounce-recovery
   ```

2. Install dependencies

   ```bash
   yarn install
   ```

3. Install and run mongodb through docker

   ```bash
   yarn start:db
   ```

4. Modify and update .env 

5. To run the api

   ```bash
   yarn dev
   ```
